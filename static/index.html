Gestion d'Entreprise

Application de Gestion d'Entreprise

Une solution complète pour gérer vos factures, projets et matériaux

Note: Cette application nécessite un serveur Python pour fonctionner pleinement.

Netlify héberge actuellement uniquement les fichiers statiques de l'application.

Gestion des Factures

Créez, modifiez et suivez vos factures clients et fournisseurs. Exportez en PDF pour un partage facile.

Gestion de Projets

Suivez la progression de vos projets, gérez les tâches et visualisez l'avancement avec des indicateurs clairs.

Tableau de Bord

Visualisez vos données importantes en un coup d'œil avec un tableau de bord intuitif et informatif.

Comment exécuter l'application complète

  1. Clonez le dépôt sur votre machine locale
  2. Installez les dépendances avec pip install -r requirements.txt
  3. Configurez Supabase selon les instructions dans SUPABASE_SETUP.md
  4. Lancez l'application avec uvicorn main:app --reload
  5. Accédez à l'application via http://localhost:8000
